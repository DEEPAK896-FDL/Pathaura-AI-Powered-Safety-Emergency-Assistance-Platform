🔌 API ENDPOINTS (25+ Total):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Authentication (4 endpoints)
  POST   /api/auth/register
  POST   /api/auth/verify-otp
  POST   /api/auth/update-profile
  GET    /api/auth/profile/:mobile
  POST   /api/auth/logout

SOS Alerts (5 endpoints)
  POST   /api/sos/trigger
  POST   /api/sos/resolve
  GET    /api/sos/active/:userId
  GET    /api/sos/history/:userId
  GET    /api/sos/unsafe-zones

Tracking (5 endpoints)
  POST   /api/tracking/start
  POST   /api/tracking/add-point
  POST   /api/tracking/end
  GET    /api/tracking/active/:userId
  GET    /api/tracking/history/:userId

Volunteers (4 endpoints)
  POST   /api/volunteers/register
  POST   /api/volunteers/verify-otp
  POST   /api/volunteers/update-location
  GET    /api/volunteers/nearby
  GET    /api/volunteers/profile/:mobile

Emergency Contacts (3 endpoints)
  POST   /api/emergency-contacts/add
  GET    /api/emergency-contacts/:mobile
  DELETE /api/emergency-contacts/remove/:mobile/:contact

Services (4 endpoints)
  GET    /api/services/nearby
  GET    /api/services/category/:category
  GET    /api/services/all
  POST   /api/services/add
